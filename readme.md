# Information Retrieval

This project implements basic information retrieval techniques. It includes code for indexing, searching, and evaluating text data.


## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Deeks-source/Information_retrieval.git
   ```
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the main script:
   ```bash
   python main.py
   ```

## Folder Structure

- `src/` - Source code files
- `data/` - Sample datasets
- `README.md` - Project documentation
